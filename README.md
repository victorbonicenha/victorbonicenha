## 👨‍💻 Sobre mim

</div>

Sou **Desenvolvedor Python** com experiência prática em **RPA, automação inteligente e IA aplicada**, atuando na criação de robôs e pipelines utilizados em processos reais de negócio.

Tenho trabalhado com automações em produção usando **Python, Playwright, Selenium, SQL Server, Linux, systemd, Telegram Bot API e integrações com IA**, construindo soluções com foco em estabilidade, rastreabilidade, monitoramento e redução de tarefas manuais.

Atualmente estou em **transição ativa para Engenharia de Dados**, aprofundando meus estudos e projetos em **PySpark, Databricks, Delta Lake, Airflow, dbt, PostgreSQL, Data Lake e arquitetura Medallion**.

Meu foco é unir minha base prática em automação com Engenharia de Dados para construir pipelines mais confiáveis, escaláveis e preparados para ambientes reais.
